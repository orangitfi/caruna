# Google Drive Usage – Quick Guide

---

## Always Start in Shared Drives

**DO:**

- Go to **Google Drive**
- Navigate to the correct **Shared Drive / team folder**
- Click **New → Docs / Sheets / Slides**

**DON’T:**

- Create files from Docs, Sheets, or Slides homepages

Note: Files created outside Drive are saved to **My Drive (personal)**

---

## Meeting Notes

**DO:**

- Store notes in a **Shared Drive folder**
- Attach an **existing file** to calendar invites

**DON’T:**

- Click **“Create”** in calendar events (creates personal files)

---

## Why This Matters

Files in **My Drive**:

- May not be accessible to others
- Can be lost during offboarding
- Create ownership confusion

Files in **Shared Drives**:

- Owned by the organization
- Accessible to the team
- Safe long-term storage

---

## Offboarding Checklist

Before leaving a project or OrangIT:

- Move all files from **My Drive → Shared Drive**
- Verify correct project/team folder
- Ensure nothing critical remains in personal ownership

---

## Quick Rules

- Start from **Shared Drive**
- Avoid using **My Drive** for shared work
- Attach files — don’t create from Calendar
- Move everything before offboarding

---

If in doubt: if others need it, it belongs in a Shared Drive
