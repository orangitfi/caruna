# Architecture Diagrams

This directory contains Mermaid diagrams that visualize the infrastructure architecture. These diagrams are written in Mermaid syntax and can be rendered by various tools including GitHub, GitLab, and VS Code.

## Diagrams

1. `architecture.mmd` - Overall infrastructure architecture
2. `security.mmd` - Security architecture and components
3. `monitoring.mmd` - Monitoring and cost management architecture

## Viewing the Diagrams

### GitHub
GitHub natively supports Mermaid diagrams. Simply view the `.mmd` files directly in GitHub.

### VS Code
1. Install the "Markdown Preview Mermaid Support" extension
2. Open the `.mmd` files
3. Use the "Markdown: Open Preview" command

### Online Mermaid Editor
1. Visit [Mermaid Live Editor](https://mermaid.live)
2. Copy the contents of any `.mmd` file
3. Paste into the editor to view and edit

## Updating Diagrams

To update the diagrams:

1. Edit the corresponding `.mmd` file
2. Use Mermaid syntax to modify the diagram
3. Preview changes using one of the methods above
4. Commit and push changes
